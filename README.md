## 🚀 O Projeto

O **Pecadômetro Diário** é uma plataforma de monitoramento comportamental que utiliza conceitos de **Machine Learning Engineering** para gamificar o registro de "deslizes" cotidianos. O projeto nasceu da necessidade de unir tecnologia robusta (React 19, Firebase) com uma interface moderna e humorística.

### 🔥 Funcionalidades Principais

-   **Hero Cards Interactive:** Interface baseada em Glassmorphism para seleção de pecados com descrições cômicas.
-   **Fate Meter (Header Dinâmico):** Um cabeçalho onisciente que reage em tempo real a cada clique, mudando de cor e veredito conforme sua alma "esquenta".
-   **Manchetes da Infâmia:** Geração automática de manchetes de "tabloide" para cada registro diário, salvas no histórico.
-   **Veredito por IA:** Modal de sucesso com sínteses dinâmicas baseadas na intensidade do seu dia.

---

## 🧠 Inteligência Artificial & Dados

Este projeto não é apenas uma lista de tarefas. Por baixo do capô, utilizamos lógica de **Data Science** para prover insights reais:

-   **Tendência Linear (Regressão):** Utilizamos OLS (*Ordinary Least Squares*) para projetar se o usuário está em uma curva de santidade ou descida acelerada.
-   **Detecção de Anomalias (Z-Score):** Identificamos dias atípicos onde o comportamento do usuário desviou significativamente da sua média histórica.
-   **Clustering Comportamental:** Algoritmo de categorização que agrupa usuários em arquétipos como *Hedonista*, *Impulsivo* ou *Egoico* com base na frequência dos pecados.

## 🛠️ Stack Tecnológica

-   **Frontend:** React 19, Vite, TypeScript.
-   **Styling:** Tailwind CSS 4.0, Framer Motion (Animações).
-   **Backend:** Firebase Firestore (Banco NoSQL em tempo real).
-   **Authentication:** Firebase Auth (Google Login).
-   **Charts/Analytics:** Recharts.
-   **Deployment:** Firebase Hosting.

## ⚙️ Como Rodar Localmente

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/jpscard/pecadometro.git
    cd pecadometro
    ```

2.  **Instale as dependências:**
    ```bash
    npm install
    ```

3.  **Configure as variáveis de ambiente:**
    Crie um arquivo `.env` ou configure no `src/firebase.ts` as suas credenciais do Firebase.

4.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```
